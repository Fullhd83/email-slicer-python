# Email Slicer (Python)

This is my first Python project.

## Features
- Validates email format
- Extracts username and domain
- Handles edge cases (multiple @, spaces, missing parts)
- Allows repeated input with user-controlled loop

## Example

**Input:**
john@gmail.com

**Output:**
username: john  
domain: gmail.com
