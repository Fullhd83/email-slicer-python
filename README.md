# Email Slicer (Python)

This is my first Python project.

## Features
- Validates email format
- Extracts username and domain
- Handles edge cases (multiple @, spaces, missing parts)

## Example

Input:
john@gmail.com

Output:
username: john  
domain: gmail.com

## What I learned
- String manipulation
- Input validation
- Basic Python logic (if/elif/else)
